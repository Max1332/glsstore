# GLS Store

Лендинг магазина спортивного питания GLS Store (Wildberries и Ozon) в стилистике игровых новостей.

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.

## Стек

- React 18
- Vite 5
- OGL (WebGL) — фон Aurora

Ссылки на Wildberries и Ozon в коде — заглушки. Замените на реальные URL магазина GLS Store.
